# Source Hierarchy Conflicts

Fandom wikis define source priority rules to resolve contradictions. A typical hierarchy:

1. **Primary canon** (films, TV episodes, official statements)
2. **Secondary canon** (licensed comics, novels, games)
3. **Tertiary canon / Legends** (superseded or alternate continuity)

## How Conflicts Arise

- A retcon contradicts a primary canon source — which takes priority?
- A new film reclassifies a book series from canon to Legends — but the wiki hasn't updated
- Two sources at the same hierarchy level contradict each other
- An editor argues that a Word of God statement (author interview) overrides published material

## Observable Signals

| Signal | What Refract Sees |
|---|---|
| **Hierarchy conflict** | Two citations at the same priority level that contradict on a factual claim |
| **Reclassification event** | A page or section moved from "Canon" to "Legends" (or vice versa) |
| **Source downgrade** | A citation replaced with a lower-priority source |
| **Interpretation drift** | Section wording changes over time without explicit citation changes |

## Example

On the Wookieepedia page for "Mandalorian culture":

- Revision 1 cites *The Clone Wars* S2E12 (primary canon)
- Revision 2 adds a contradictory detail citing a 2019 reference book (secondary canon)
- Revision 3 removes the secondary canon citation, reverts to Revision 1 language
- Revision 4 reintroduces the claim but with a "Legends" tag

Refract reports: *Source hierarchy conflict detected. Claim about Mandalorian succession rules has alternated between primary and secondary canon citations across 4 revisions. Dispute present on talk page.*

## Not Tracked

- Which source hierarchy is "correct"
- Whether a retcon was justified
- Fan consensus about which interpretation is better
